- 【中文】
- 你将使用到的硬件有：
- Arduino UNO主板
- ADXL335 3-Axis三轴加速度传感器
- 串口数据线
- 
- 连接方式为：
- 	X -- A2
- 	Y -- A1
- 	Z -- A0
- 	+ -- 5V
- 	- -- GND

- [English]
- The hardware you need:
- Arduino UNO board
- ADXL335 3-Axis three axis acceleration sensor
- serial cable
- 
- Connection:
- 	X -- A2
- 	Y -- A1
- 	Z -- A0
- 	+ -- 5V
- 	- -- GND
